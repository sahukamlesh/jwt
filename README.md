
# jwt authentication app




## Run Locally 

Clone the project

```bash
  git clone https://github.com/sahukamlesh/jwt.git
```
To run locally client file 

```bash
  cd client 
```

Install dependencies

```bash
  npm install
```

Start the client file by using this command 

```bash
  npm run start
```
 ######################## To run locally Server  File #####
```bash
  cd server 
```
install dependencies


```bash
  npm install
```
to start the server 

```bash 
npm run dev 
``````
